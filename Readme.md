## Build Instructions

The entry point is Main.java in the project2 package. Import the project into your preferred IDE and run Main.java, or with the command `java project2\Main.java`

This project requires the Google Guava library. Make sure it is added to your classpath before running. Additionally, the testing suite relies on JUnit5.
