# Project Scope
The project’s core goal is to demonstrate set and multiset operations using a defined universal set as the domain, and to make the operations easy to follow and understand. 

## Project Features

We have a UniversalSet from a list of string elements.
The class OrdinarySets is represented as a boolean bit array aligned to that universe, and performs standard set operations. 
Complement, union, intersection, difference and symmetric difference are all built as methods within the class.
The class MultiSets tracks how many times each element appears and performs multiset-specific operations.
Union, intersection, difference, and sum are methods in this class. 
The Main class is the driver for the project, and demonstrates cleanly and clearly the operations with labeled output. 

## Project Constraints

The program is hardcoded and does not allow input from the users. 

It does not support element types beyond Strings

It does not allow for resizing of the universal set after it is created. 

It cannot whip up a Margarita or Old Fashioned for you, although it does know what is in these cocktails.
